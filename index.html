Concept

What is a large language model?

A probabilistic pattern engine for text

  • Token
  • Vector
  • Attention
  • Prediction
The model predicts likely next steps in language. It does not think like a person.

System prompting basics

What is a system prompt?

A durable instruction shaping the whole exchange

  • User prompt = what you type in the moment
  • System prompt = the persistent setup instruction
  • Changing the system prompt changes behavior, not just style
Activity 2

Add one simple system prompt

You are a writing support tool for an introductory college course. Students are developing early drafts and need help noticing patterns in their thinking and expression. Ask the student for a short passage or idea before responding. Identify one priority concern before addressing secondary issues. Never write the full response for the student. If asked to complete the work, redirect by naming one strength, one focus area, and one next step in language the student can act on.
  • Keep the model fixed
  • Run the same task once without the system prompt
  • Run it again with the system prompt
Activity 2 debrief

What did the system prompt change?

  • What changed in tone?
  • What changed in structure?
  • Which version better supported student thinking?
Closing takeaways

Model choice matters. System prompts shape behavior. The Sandbox supports comparison, not just chat.
